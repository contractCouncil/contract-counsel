# ContractConsel

AI-powered legal technology platform redefining how legal work gets done.

## About

ContractConsel is an AI company that empowers lawyers and legal professionals to operate at peak accuracy while dramatically cutting the time spent on routine legal tasks. We sit at the intersection of artificial intelligence and legal services.

## Services

- **Contract Review** - AI-powered risk identification and clause analysis
- **Contract Drafting** - Jurisdiction-aware contract generation from refined templates
- **Document Review** - Intelligent categorization, prioritization, and flagging
- **Legal Documentation** - Automated creation and management with compliance assurance
- **eDiscovery** - AI-driven search, predictive coding, and document clustering

## Tech Stack

- **Framework:** Next.js 16 (App Router, TypeScript)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Smooth Scroll:** Lenis

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```
app/              # Next.js app router pages and global styles
components/
  layout/         # Navbar, Footer
  sections/       # Hero, Services, About, Contact
  ui/             # Reusable components (Button, AnimatedSection, etc.)
lib/              # Constants and site configuration
public/           # Static assets (logo, images)
```
